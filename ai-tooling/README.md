\#ai-tooling

Scripts for detecting and managing AI tooling deployed to endpoints, including silently installing model weights, browser-based AI features, and AI coding assistants.

As AI tooling becomes embedded in standard software distributions, security teams need visibility into what is being installed on managed endpoints without explicit consent or notification.

## Scripts in this category

| Script | Platform | Purpose |

|--------|----------|---------|

| detect-gemini-nano | macOS, Windows | Detect Google Chrome Gemini Nano model weights |

## Compliance relevance

Unauthorized or undisclosed software installations, expansions, or add-ons are relevant to:



\- \*\*SOC 2 CC6.8\*\* - Controls to prevent unauthorized software

\- \*\*PCI DSS 6.3.3\*\* - All system components protected from known vulnerabilities

\- \*\*NIST CSF DE.CM-7\*\* - Monitoring for unauthorized personnel, connections, devices, and software

\- \*\*ISO 27001 A.12.6\*\* - Management of technical vulnerabilities
